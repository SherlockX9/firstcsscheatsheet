Cheatsheet on CSS selectors and specificity

On this table you'll find the various forms of CSS selectors, what they mean, and how to use them. With examples, of course!

Selector type Creating selector Accessing selector Selector Specifity
Everything in the html doc: '*' No specific HTML required for this, simply use: *{css-styling here} to apply to the whole HTML doc Least specific, only applies if the rest of the selectors shown aren't used
Class <tag class = "classname"> .classname{css-styling here} Overrides the '*' selector
Element tags Element itself is the selector: e.g <p>,<footer> .elementname{css-styling here} Overrides the '*' and Class selector
ID selectors <tag id = "yourIDname"> #idname{css-styling here} Overrides any other selector
Elements within a class Say you have <p> tag within a class called 'sectionOne': .sectionOne p{css-styling here} Overrides all selectors except the id selector
